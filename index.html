Shannz Rest APIs

Shannz rest api adalah website penyedia endpoint api dari berbagai jenis kategori yang pastinya gratis.

Saya juga menyediakan simple code case agar memudahkan kalian dalam mengimplementasikan api ini kedalam bot whatsapp kalian.

Jadi tunggu apalagi? segera pakai dan gunakan, ambil semuanya hanya dengan result json

/ai
/simsimi?
https://shannmoderz-95f1d384b6d2.herokuapp.com/ai/simsimi?query=
parameter required
/simsimi?query=
simple case code
case 'simi': {
  if (!text) return m.reply('masukkan pertanyaanmu!')
  try {
    let simsimi = await (await fetch(`https://shannmoderz-95f1d384b6d2.herokuapp.com/ai/simsimi?query=${text}`)).json()
    let shannz = simsimi.result
    m.reply(shannz)
  } catch (err) {
      m.reply('masukkan query lainnya atau coba lagi nanti')
    }
}
break
